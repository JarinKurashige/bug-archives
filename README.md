# bug-archives

## Build app

`make build`

## Run app

`make new_bug`

Fill out the form according to instructions.

![](https://github.com/JarinKurashige/bug-archives/blob/0cafe53b9155bb979da9cb0c16337bcdabcc9bda/media/Fill%20Out%20Form.png)

If at any point you mess up, you can either quit and restart, or the form will confirm that all fields are correct when completed.

![](https://github.com/JarinKurashige/bug-archives/blob/0cafe53b9155bb979da9cb0c16337bcdabcc9bda/media/Confirm%20and%20Save%20Form.png)
